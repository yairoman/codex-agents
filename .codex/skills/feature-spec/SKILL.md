# Feature Specification

## Goal

Transform a vague request into a clear implementation spec.

## Rules

- Clarify scope
- Define acceptance criteria
- Identify edge cases
- Capture assumptions

## Output

1. Problem statement
2. Scope
3. Out of scope
4. Acceptance criteria
5. Edge cases
6. Assumptions
